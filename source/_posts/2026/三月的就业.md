好久没写文章了，上次长篇大论还是去年年末。我的写作水平退化到了什么程度？恐怕初中生考试作文也比我强上不少。哎呀，从哪说起呢，这三个月经历的事情还真不少。最大的变化莫过于周遭环境，不再是我生活多年的小城市。没错，我搬家了。算是搬家吧，贵重物品还放在老家没带走，但我身心已经决定不再驻留于那个小城市。小城市有亲爱的爷爷奶奶，大城市有我所不知的见识，两者不可兼得。爷爷奶奶终有一日会离我而去，与其眼睁睁地看见残酷的现实，不如选择主动离开。

这个决定，从去年一直纠结到了今年过年，直到老妈回了老家，我不得不跟着她一块去大城市时才动身前往。还记得那天是大年初四的清晨，我背上背包，拉起行李箱，在小区门口向爷爷奶奶告别。这一离开，真不知道何时才能再见面。路途中一直思考人生和未来。未知的未来，犹如在 Minecraft 中探索未知结构，充满好奇与担忧。我不觉得现实世界有多有趣，对于一个在这里生活了二十有余年头的人来说已不再新奇，发生的事情往往不尽人意。

大城市确实有它的魅力，不断吸引年轻人聚集，从中诞生新鲜知识和故事，遇见不同的人。我偶尔如现在这般充满对学识的疯狂渴求，偶尔又万念俱灰生不如死。我一直在问自己到底需要什么才能满足，到底追求着什么弥补心灵空缺？大脑一片空白。

刚刚一位妇人骑电动车来到庭院边，将狗拴在离我约莫三米的树干下，遗弃般离去，任由它不断朝我吠叫，吵得我脑壳痛。如此处境怎能平复心情写作？这狗，为何朝我叫个不停？一没得罪它，二没影响它，三没在此地移动半步，它偏要不断叫喊，宣告自己拥有此地主权般恶狠狠盯着我叫。罢了罢了，不跟狗类做无谓的较劲，另寻一处安然坐下。

来X市已有一个月之久，这个月里的大部分时间是在找工作和面试（约到面试的次数一只手就能数完），其余时间用在探索地图、复习数据库、玩 Minecraft 上了。没入职前很焦虑很烦躁，每天早上还没睁眼老妈就喊我起床去找工作，吃饭也念叨此事，每逢店里有来客便上前询问是否有工作推荐，你说这不烦吗？她找工作的焦虑甚至在我之上。最后靠自己找到了现在这个月薪三千的工作。准确讲是四千，交完社保就剩三千了。相比于上一份月薪七八千的工作，这差距属实有点大。不过怎么说呢，有双休，早九晚六这两点确实不错。出差的话，能增长点见识也是好事吧（大概）。除了离家太远这个最大的问题，路程花费时间在一小时多一点，交通费包括地铁五块和公交两块，一天下来就是十四块。中午吃饭开销在十六块左右，也就是说每天基础消费是三十块，一个月按三十天计算，就是九百块固定开销。这样一来，净收入就只有两千多一点而已，少得可怜啊。

那天面试我的人好像是这家公司的老总，早上面完下午人事（也是经理）发通知让我准备一些材料下周二早上九点到公司报到。看到通知的那会儿相当激动啊，脑袋晕乎乎的，可转念一想这公司试用期的条件其实不比在老家随便找个两三千的工作强。说是正式员工会加工资且有绩效奖励，老板画的饼谁知道是不是真的。这公司总共十五人不到，其中还有两三个实习生。我面试的岗位是技术支持，入职后实际岗位是实施。公司经营的主要是地方行政单位安装智能监控系统，所以必须出差。出差地点大多在省内，部分在外省乃至全国。签合同还得签保密协议，目前最大的疑虑是出差到项目地点驻场可能不能带电子设备进入，或是会检查随身携带的设备。而且项目地点绝大多数在郊区或县城附近，没有娱乐生活设施，住宿是与甲方员工一同住集体宿舍。听销售说，下周可能安排我与同事出差。

要不是能出差做项目见见世面，天天待在办公室标注图片我早就跑路了。据说标注图片时会收手机，领导们担心有人拍照标注内容发网上博眼球，由于我来的这几天领导出差，所以手机自由使用。我倒觉得想拍照的人早就乘着领导出差拍完了，哪还等着在领导眼皮子底下拍照。

况兄比我早一周找到工作，他说现在在H市一个超大的外包公司给华为平板做测试，美名其曰【测试工程师】实际只是个点点点工程师，工资 4.4k 封顶，试用期拿 90%，早九晚六，双休但是每月固定有一天加班，交完社保到手的也跟我差不多。他还在边上稍远的地方租了个单间，有独卫有阳台，环境干净整洁，比之前合租的条件好多了，只是房租有七百多。他在外吃饭，所以一个月下来净收入基本为零。

况兄有大半年没工作了，而我则是一年哈哈，算下来去年三月底离职今年三月底上班正好一年呢。当然，在简历里写的是去年年底离职，面试就说项目结束正好回家过年，过完年来大城市找工作。

工作了一周，让我感觉开心的只有周三那天给我培训服务器命令行使用。俺可是计算机专业出身，更何况是最最最熟悉的 Linux 操作系统。看见真正的 1U 服务器，还专门拆开顶盖给我看了看内部构造，可算是大饱眼福啊。这是一台 aarch64 架构的服务器，除了 Debian12 主系统外，还有四个子系统 Ubuntu 24.04。主系统与子系统之间不共享硬件，也就是说服务器内部有五个 SoC，五个 16G 内存和五个 256G eMMC。SoC 和主板只用散热片盖住（我看到的是四个，还有一个不知道在哪），在电源附近的是一排风扇。由于是 aarch64 架构，不像 AMD64 那般可以自由安装系统，只能用厂商定制内核（记得好像是 6.1）。这种服务器只能使用服务器厂商提供的系统固件，主系统用调试接口刷机，子系统用主系统自带管理页面刷机。四个子系统安装相同的 Java 后端服务用于负载均衡，主系统安装 MySQL、Redis 和 Nginx。除了服务器本身管理外，还给我展示了项目本身的网页和运维页面。其实就是个很小的项目，跟之前工厂里用的管理系统相比真的很小。难怪只有两个 Java 后端和一个前端。

不知道这份工作能干多久。第一天让我干数据标注的时候就想离职了，明明早上才签完合同。

一不留神写了好多工作上的事呢。今天周末，明天又是工作日了。再说说家里宽带的事情吧。我房间的宽带一周前因突然跳闸停电导致断网，折腾了好几天最后叫联通师傅昨天来检修才发现可能运营商那边改了光猫认证配置，导致猫棒认证失败无法联网，最后用原本的光猫桥接路由器拨号才得以成功上网。今天再登录家里云盘发现又无法连接，恐怕这次还是跳闸停电吧。哎呀我在家都没这么多事怎么一离开就出各种问题，烦死了。

以后的休息日，要不要考虑做点项目练手？
